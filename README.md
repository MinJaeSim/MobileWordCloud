# Mobile Word Cloud
1. 웹 사이트를 스크레이핑하여 단어 빈도수 카운팅하고 그 데이터를 바탕으로 키워드 시각화

2. 구글 드라이브에 있는 카카오톡 대화록 텍스트 파일을 분석하여 단어 별 빈도수를 카운팅하고 그 데이터를 바탕으로 키워드 시각화
